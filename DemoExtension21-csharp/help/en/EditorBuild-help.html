Build Editor

Once your pick operation and place layer jobs have been created, you can define how the jobs are assembled together to form a stack of layers on a pallet at your build stations. 

First, select a build station, the box product being placed on the pallet and the pallet being used.  Once selected, a list of layer jobs will appear.  Select the layer to be added from the list. If the product of that layer has multiple pick operations defined for it, a list of the possible pick operations will appear too.  The selected layer job can be added to the stack by pressing the Add Layer on Top button.  The layer appears in the table below representing the stack of layers.  Layers from stack can be removed using the Delete Selected Layer button.

Once a build pattern is complete, press the Save & Generate Job button.

Note: Pallets handled by your robot can only be placed as the first layer of the stack.
