The PalletBuilder extension is a graphical interface to quickly generate jobs for palletization.  You may use the 'Guided Build Pattern Creation' option to create a palletization pattern with step-by-step guidance.  The tabs on the side can be used to access, create or modify the components of a palletization operation. 

Tab Guide:

  • Home: Return to the home screen of the extension.
  • Cell Operation: Run your cell and monitor status.
  • Manage Products: Define and edit the products used in a palletization cell.
  • Pick Operations: Create or edit jobs to pick your products.
  • Place Layers: Create and edit jobs to place your products on a layer.
  • Build Patterns: Define and edit how the layers will be assembled on the pallet.
  • Cell Setup: Define the stations and grippers of your cell and adjust general settings of your palletization operation. Save and load a PalletBuilder database.
