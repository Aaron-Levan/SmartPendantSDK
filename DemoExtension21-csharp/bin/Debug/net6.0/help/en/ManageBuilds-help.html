Manage Builds

View a list of build pattern jobs.  Select a job from the list and press the Edit button to modify it. Create a new build pattern job using the New button. 

Job Statuses:

  • OK: The job is acceptable and ready to use. 
  • Missing: The job is no longer on the controller. 
  • Unrelated: The job exists on the controller but it is not a job generated by PalletBuilder or the job tag is different from "PALLETBUILDER". 
  • Modified: The job was modified outside of PalletBuilder. 
  • Old: The job on the controller is older than the one noted in the database. An older version might have been loaded in the controller. 

Generally, you may want to generate jobs that do not have an OK status.  This can be done using the Generate Job button.  Jobs can be deleted or copied using the Delete button or Duplicate buttons respectively.  Duplicate buttons respectively.
