Pick Operation/Place Layers

This screen allows you to create jobs to pick up or place products. 

Initially, you need to assign a name, station and product.  This will construct a job that will dictate how the selected product is handled.  For pick jobs, only infeed stations and dispenser type stations are selectable.  Place jobs will only give access to build stations. 

PalletBuilder offers the ability to make jobs generic by selecting the Used on similar station checkbox.  This will allow the job to be used on another station of the same type with respect to its user frame.  This feature requires that the positioning of the product relative to the user frame be the same for all similar stations.  For example, if one infeed is aligning product on the left side and the another on the right, this feature cannot be used.  Without this condition, each station will require a unique job as seen in the following figure.

   

For the product, select a product from the list. If a pallet is to be handled by the robot, the pallet must have a gripper selected in the "Product Manager" screen in order to display it in the product list. For the Place Layer job, a pallet must also be selected to size the layer correctly. And for a box product, the Layer Design button will be available to define the boxes layout and sequence of placement. The place path will need to be programmed only for the first box of the layer.

Before proceeding to program a path for the robot for the job, you may wish to configure number of points in the path.  PalletBuilder allows you to change the number of Approach or Departure points.  This can be used if you need a more complex path to position the product or avoid collisions with neighboring products.  For simpler paths, PalletBuilder also provides an option to use the same path for approach and depart motions using the Invert Approach for Depart Path checkbox.  With this option enabled, only the approach and pick/place points need to be defined.  The departure points will be reflected about the pick/place point to create symmetry in the motion. 

A default path will be generated based on the selected station and product.  That path will need to be verified by pressing the Edit Motion button and going through each point. You may press the Reset Path button to initialize your path and path configuration back to the default path.

Notes:

  • Invoking the invert option will cause the number of approach and departure points to match. 
  • If an approach point was taught, adding extra approach points will produce copies of the taught approach point.  These new points can be overwritten. 
  • If a departure point was taught, adding extra departure points will produce copies of the taught departure point.  These new points can be overwritten. 

Once completed, press the Save & Generate Job button to save the job or press Cancel to discard the job or changes.
