ステーション

ステーションは、パレタイズセルの要素です。 

すべてのステーションには、ユーザ座標系が関連付けられています。  ユーザ座標系は、歯車アイコンを押してアクセスできる、スマートペンダントユーザ座標系メニューを使用して設定できます。 

教示する場合、教示位置は割り当てられたユーザ座標系を基準とします。  適切に操作するには、各ステーションタイプの推奨位置でユーザ座標系を設定することが重要です。  特定のステーションタイプのユーザ座標系を教示する方法の詳細は、[ユーザ座標系]ドロップダウンリストの横にある情報アイコンを押してください。 

変数の割り当てセクションで、そのステーションのパレタイズジョブが参照する変数を割り当てる必要があります。  変数の割り当てに続いて、ステーションに制御を追加するためのI/Oを選択できます。  これらのI/Oを使用して、ステーションの準備が整ったとき、リセットが必要なとき、または積み付けが完了したとき、コントローラに通知できます。 

必要に応じて、ステーション間を移動するときに呼び出されるジョブを割り当てることもできます。  これらのジョブを使用して、ステーション間の特定の経路に沿ってロボットを移動させ、フェンス、支柱、または他のステーションなどの障害物を回避できます。  また、グリッパーやその他のデバイスへのI/O信号を有効にするためにも使用できます。 

各ステーション特性:

積付ステーション:

  • 積付パターン S変数: 割り当てられた積付パターンのジョブ名を格納するために使用される文字列変数。 
  • 積付け高さ D変数: 積付 ステーションの現在の高さ (ミクロン単位) を追跡し、それに応じて段をシフトするために使用される倍精度整数変数。
  • 積みレイアウト(段) I変数: 積付の現在のレイアウト(段)番号を持つ整数変数 (パレットとセパレーターもレイアウト(段)としてカウントされます)。
  • サイクル I変数: レイアウト(段)の、現在のサイクル (製品(箱)) 番号を持つ整数変数。 
  • レディー入力信号: 積付ステーションの準備が整ったことを確認する入力信号。  多くの場合、パレット存在センサーに関連付けられています。 
  • ステーション出力信号: ロボットが現在動作しているステーションの出力信号。 
  • リセット入力信号: 積付ステーションの積付高さを 0 、レイアウト(段)変数とサイクル変数を 1 にリセットする入力信号。 
  • 完了出力信号: 現在の積付が完了したことを示す出力信号。 

注: 通常、完了出力信号とリセット入力信号は一緒に機能します。積付パターンが完了すると、完了出力信号がオンになり、パレットが取り外されて新しいパレットが所定の位置に配置されると、リセット入力がオンになります。完了信号がオフになると、リセット入力をオフにすることができます。

供給ステーション:

  • 位置合わせ: 供給製品(箱)がコンベアの右、左、中央のどこに配置されるかを示します。 
  • 幅: コンベアの幅 (mm)。 ユーザ座標系に対する製品(箱)の位置を計算するために使用されます。 
  • 製品 S変数: 供給に割り当てられた製品名を格納するために使用される文字列変数。 
  • レディー入力信号: 供給ステーションの準備が整ったことを確認する入力信号。  多くの場合、コンベアの端にある製品(箱)の存在センサーに関連付けます。 
  • ステーション出力信号: ロボットが現在動作しているステーションの出力信号。 

ディスペンサ(固定位置):

  • 製品 S変数: ディスペンサーに割り当てられた製品名を格納するために使用される文字列変数。 
  • レディー入力信号: ディスペンサー ステーションの準備が整ったことを確認する入力信号。 多くの場合、製品の存在を検出するセンサーに関連しています。 
  • ステーション出力信号: ロボットが現在このステーションにいることを示す出力信号。

ディスペンサ(センサ使用):

  • 高さ: ステーションに出入りするときにクリアする必要があるディスペンサーの側面の高さ。 
  • 製品 S変数: ディスペンサーに割り当てられた製品名を格納するために使用される文字列変数。 
  • サーチ調整 D変数: 最後に見つかった製品の場所を追跡するために使用される倍精度型変数。  有効にすると、スタックの最大高さからサーチを開始するのではなく、最後に見つかった場所から次のサーチ動作が開始されます。
  • レディー入力信号: ディスペンサーステーションの準備が整ったことを確認する入力信号。  多くの場合、製品の存在を検出するセンサーに関連しています。 
  • ステーション出力信号: ロボットが現在このステーションにいることを示す出力信号。
  • リセット入力信号: ディスペンサーがスタック空出力信号をリセットし、サーチ調整変数の高さを 0 にリセットする入力信号。
  • スタック空出力信号: サーチが前回のサーチで教えられたピック位置に到達したことを示す出力信号。
  • サーチ入力信号: 製品の存在を検出するグリッパーからの入力信号。  [サーチの開始時にツールを有効にする]オプションがあります。これにより、真空確認センサーがサーチ入力として使用されている場合に、サーチの開始時に真空をオンにすることができます。 
  • 製品が薄く、「サーチ調整」変数が有効になっている場合、「最後に見つかった部品の Z mm 上で次の検索を開始する」に距離を追加できます。
  • スタック少量信号: 有効にすると、検出された製品が指定された値 (mm) を下回ると、指定された出力信号がオンになります。 

注: 通常、スタック空出力信号とリセット入力信号は一緒に機能します。  ディスペンサーが空になると、スタック空出力信号がオンになり、ディスペンサーが再装填されると、リセット入力がオンになります。  完了信号がオフになると、リセット入力をオフにすることができます。
