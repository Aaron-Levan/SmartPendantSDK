ジョブの管理

取り/置きジョブリストを表示します。  リストからジョブを選択し、編集 ボタンを押してジョブを編集するか、新規ボタンで、新しいジョブを作成します。  ジョブが作成されると、リストに表示されます。 ジョブのステータスは、ジョブステータス列に反映されます。   ジョブのステーションが削除されている場合、ステーションは[(未定義)]と表示されます

ジョブステータス:

  • OK: 使用する準備ができています。
  • 不明: ジョブがコントローラ上にありません。
  • 無関係: ジョブはコントローラ上に存在しますが、パレットビルダーによって生成されたものではないか、ジョブタグが［PALLETBUILDER］とは異なります。 
  • 変更: パレットビルダーの外部でジョブが変更されました。
  • 旧: コントローラ上のジョブは、データベースに記録されているものよりも古いものです.  古いバージョンがコントローラに読み込まれている可能性があります。 

OK ステータスを持たないジョブを生成したい場合は、ジョブの生成ボタンを使用して実行できます。  ジョブは、削除 ボタンまたは 複製 ボタンをそれぞれ使用して削除またはコピーできます。
